# Changelog

All notable changes to DuoAsk will be documented in this file.

## Unreleased

- No unreleased changes yet.

## 0.2.0 Multilingual UI

- Add a 中文 / English interface switch with local preference persistence.
- Localize the main app shell, sidebar, status labels, history view, logs, and result states.
- Update answer and synthesis cards so status labels can use the active language.
- Refresh README positioning with bilingual instructions and privacy notes.
- Replace README screenshots with mock-data captures that do not contain real prompts, accounts, cookies, or chat history.

## 0.1.0 MVP

- Improve README positioning for public open-source launch.
- Add community support, conduct, and dependency maintenance files.
- Polish SMTP settings copy to avoid personal environment references.
- Add Windows portable zip packaging script.
- Add GitHub social preview artwork.
- Add README workflow demo GIF.
- Electron + React + TypeScript desktop app.
- Browser-based provider adapters for ChatGPT, Claude, Gemini, Kimi, Doubao, and Grok.
- User-assisted login and local browser session reuse.
- Side-by-side provider answers and rule-based synthesis.
- Local HTTP API on `127.0.0.1:3719`.
- Local task history and export helpers.
- MIT license, contribution guide, security policy, issue templates, and CI.
