# Support

Thanks for trying DuoAsk.

## Questions And Usage Help

Use GitHub issues for reproducible bugs, setup problems, and feature requests. Before opening an issue, please check:

- [README.md](README.md)
- [Setup notes](docs/SETUP.md)
- [Provider strategy](docs/PROVIDERS.md)
- [Security policy](SECURITY.md)

## Before Posting Logs Or Screenshots

Do not post:

- cookies or browser profile files
- prompt history or private provider answers
- API tokens, SMTP credentials, or `.env` files
- account identifiers, billing pages, or verification screenshots
- provider pages that expose personal data

If a provider login or verification flow fails, describe the visible state in words and remove private details from any screenshot.

## Security Reports

Please do not open public issues for security vulnerabilities. Follow [SECURITY.md](SECURITY.md) instead.
